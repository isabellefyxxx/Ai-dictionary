## 部署到 Vercel

### 方式一：GitHub + Vercel（推荐）

1. 在 GitHub 上创建新仓库
2. 将本项目文件推送到仓库
3. 前往 [vercel.com](https://vercel.com) → Import Project
4. 选择你的 GitHub 仓库
5. 直接点 Deploy，无需任何配置

### 方式二：Vercel CLI

```bash
npm i -g vercel
cd ai-funny-dictionary
vercel
```

按提示操作，几秒即可部署完成。

### 方式三：拖拽部署

1. 前往 [vercel.com/new](https://vercel.com/new)
2. 拖拽整个项目文件夹上传
3. 自动部署
